Uses Firebase (FCM push notifications). Fonts: Poppins, Inter. Min SDK: Android 7 (API 24).

## Tech Stack
- Framework: Flutter (Dart)
- Backend: Firebase (FCM, Core)
- Fonts: Poppins (primary), Inter (secondary)
- Icons: iconsax_plus, lucide_icons_flutter, line_icons
- Min SDK: Android 7.0 (API 24)
- Target SDK: Android 16 (API 36)

## Design Rules (always follow)
- Font: Poppins for headings, Inter for body
- Always check Figma file before creating any UI
- All screens must have bottom navigation
- Follow Material Design 3 guidelines
- All UI must be accessible (WCAG AA)

1. Always check Figma status before design work (`figma_get_status`)
2. After every UI change — take a screenshot to verify (`figma_take_screenshot`)
3. Never write code without first planning the screen structure
4. Always use the `flutter-expert` agent for Flutter code
5. Commit message format: `feat:`, `fix:`, `refactor:`, `design:`
